How to Use Values Mapping

Quick Start

  • Read each ethical scenario carefully
  • Choose the response that best reflects your values
  • Complete 8-10 scenarios for a comprehensive profile
  • Download your personalized values.md file

Keyboard Controls

  • A S D F - Select choices 1, 2, 3, 4
  • 1 2 3 4 - Alternative number keys
  • ↑ ↓ - Navigate choices
  • Enter - Confirm selection
  • ? - Show this help

Content Management

The content management panel at the bottom provides several ways to add scenarios:

  • Load More: Load additional scenarios from your selected data source
  • Generate from Frameworks: Create scenarios by combining motifs and frameworks
  • Configure AI: Set up API credentials for AI-powered scenario generation
  • Generate with AI: Create new scenarios using language models

Data Sources

  • Local files: dilemmas.csv from your deployment
  • GitHub repository: Latest scenarios from the project
  • Framework generation: Combinatorial scenarios from motifs + frameworks
  • LLM generation: AI-created scenarios (requires API key)
  • Built-in scenarios: Curated ethical dilemmas

Click the data source indicator in the top-left to switch between sources or add scenarios.

Forking the Repository

To use your own dilemmas:

  1. Fork the values.md repository
  2. Add your dilemmas to striated/dilemmas.csv
  3. Deploy to GitHub Pages
  4. Your scenarios will load automatically

Results & Publishing

After completing your assessment, the results panel provides:

  • Assessment Report: Human-readable summary of your responses
  • values.md File: Formal profile for AI alignment
  • GitHub Publishing: Share your profile on GitHub
  • AI Testing: Test how well AI understands your values

Local Development

To run locally and avoid CORS issues:

Python HTTP Server

python3 -m http.server 8000 # Then open: http://localhost:8000/values-generator.html

Node.js Serve

npx serve . # Then open the provided URL

About Values.md

This tool generates personalized ethical frameworks for AI alignment. Your responses create a structured values.md file that can be used to align AI systems with your moral reasoning patterns.
