About

What is VALUES.MD?

VALUES.MD is a standardized format for defining ethical frameworks with clear hierarchies and priorities.

Structured Format
Transform abstract ethical principles into concrete decision frameworks with clear hierarchies.
Machine Readable
Easily parse and integrate ethical frameworks into software systems and AI.
Transparent Reasoning
Make ethical reasoning explicit, auditable, and communicable to stakeholders.
Adaptable Framework
Balance consistency with flexibility through domain-specific frameworks.

Documentation

Get Started with VALUES.MD

Installation

npm install values-md

Or include it directly in your HTML:

<script src="https://unpkg.com/values-md@1.0.0/lib/values-md.min.js"></script>

Basic Usage

// Parse a VALUES.MD file
import { parseValuesMD } from 'values-md';

const valuesMD = parseValuesMD(fs.readFileSync('values.md', 'utf8'));
console.log(valuesMD.coreValues);

// Evaluate a decision
const ethicsSystem = new ValuesMDSystem(valuesMD);
const evaluation = ethicsSystem.evaluateDecision(decision, context);
