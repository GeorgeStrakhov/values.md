How to Use Values Mapping

Quick Start

  • Read each ethical scenario carefully
  • Choose the response that best reflects your values
  • Complete 8-10 scenarios for a comprehensive profile
  • Download your personalized values.md file

Keyboard Controls

  • A S D F - Select choices 1, 2, 3, 4
  • 1 2 3 4 - Alternative number keys
  • ↑ ↓ - Navigate choices
  • Enter - Confirm selection
  • ? - Show this help

Data Sources

  • Local: Files from your deployment
  • Remote: GitHub repository
  • Built-in: Embedded scenarios

Local Development

To run locally and avoid CORS issues:

Python HTTP Server

python3 -m http.server 8000 # Then open: http://localhost:8000/values-generator.html

Node.js Serve

npx serve . # Then open the provided URL

About Values.md

This tool generates personalized ethical frameworks for AI alignment. Your responses create a structured values.md file that can be used to align AI systems with your moral reasoning patterns.
